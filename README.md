# 💼 Job Application Portal

A modern and responsive **Job Application Form** built using HTML, CSS.
This project focuses on clean UI design, user-friendly interaction, and basic front-end validation.

---

## 📌 Overview

This application provides a simple and efficient interface for users to submit job applications.
It includes structured input fields, file validation, and a smooth submission experience.

The goal of this project is to demonstrate strong fundamentals in:

* Frontend development
* Form handling
* UI/UX design principles

---

## ✨ Features

* Responsive design (works across devices)
* Clean and minimal user interface
* Form validation (required fields, email format)
* File upload restriction (PDF only, max 2MB)
* Auto-selected current date with validation
* Interactive input focus and hover effects
* Submission success message without page reload

---

## 🛠️ Technologies Used

* **HTML5** – Structure and layout
* **CSS3** – Styling (Flexbox & Grid)
